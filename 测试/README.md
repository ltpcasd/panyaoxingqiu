## 🚀 快速开始

### 第一步：环境准备

1. **配置后端环境变量**
   ```powershell
   cd "d:\ai项目\ai-test-demo\后端\panyaoxingqiu-server"
   # 复制模板文件
   Copy-Item .env.template .env
   # 编辑 .env 文件，填写数据库密码等配置
   ```

2. **初始化数据库**
   ```sql
   -- 在MySQL中执行
   CREATE DATABASE IF NOT EXISTS panyaoxingqiu 
   DEFAULT CHARACTER SET utf8mb4 
   COLLATE utf8mb4_unicode_ci;
   ```

3. **启动服务**
   ```powershell
   # 终端1：启动后端
   cd "d:\ai项目\ai-test-demo\后端\panyaoxingqiu-server"
   npm install
   npm run start:dev
   
   # 终端2：启动前端
   cd "d:\ai项目\ai-test-demo\前端\panyaoxingqiu-miniapp"
   npm install
   npm run dev
   ```

### 第二步：执行测试

1. **接口测试**
   - 访问 Swagger 文档：http://localhost:3000/api/v1/docs
   - 或使用 Postman 导入 `API测试集合.md` 中的内容
   - 参考 `联调测试执行指南.md` 执行测试

2. **前后端联调**
   - 参考 `联调测试检查清单.md` 逐项测试
   - 在 `测试报告-联调.md` 中记录测试结果

3. **问题记录**
   - 发现问题时在测试报告中记录
   - 定位问题归属（前端/后端）
   - 跟踪修复进度

## ⚠️ 注意事项

1. **测试数据隔离**：使用测试专用账号，避免污染生产数据
2. **接口鉴权**：大部分接口需要携带Token，先完成登录获取Token
3. **配对测试**：需要两个测试账号配合完成配对流程
4. **数据库备份**：测试前建议备份数据库
5. **日志记录**：测试过程中注意收集后端日志
