## 三、头像自定义功能设计

### 3.1 功能概述
支持用户自定义上传头像，包括：
- 双人头像（首页顶部展示）
- 个人头像（个人中心、各页面头像展示）

### 3.2 头像类型定义

#### 3.2.1 双人头像（Couple Avatar）
- **展示位置**：首页顶部、时光轴页、默契考验页等
- **尺寸要求**：建议 400×400 像素，正方形
- **格式支持**：JPG、PNG
- **文件大小**：不超过 2MB
- **默认状态**：显示系统默认情侣头像插画

#### 3.2.2 个人头像（Personal Avatar）
- **展示位置**：个人中心、设置页、消息通知等
- **尺寸要求**：建议 200×200 像素，正方形
- **格式支持**：JPG、PNG
- **文件大小**：不超过 1MB
- **默认状态**：显示用户性别对应的默认头像

### 3.3 功能入口设计

#### 入口一：个人中心 - 头像区域（快捷入口）
```
位置：个人中心页顶部用户信息区域
交互：点击当前头像 → 弹出头像设置选项
选项内容：
  - 更换我的头像
  - 更换双人头像
  - 查看大图
  - 取消
```

#### 入口二：设置页 - 头像设置（完整入口）
```
位置：个人中心 → 设置 → 个人信息设置
路径：个人中心 > 设置 > 头像设置
包含选项：
  ├─ 我的头像 [当前头像预览]
  ├─ 双人头像 [当前头像预览]
  └─ 头像历史记录
```

### 3.4 头像上传流程

#### 流程图
```
用户点击头像区域
    ↓
弹出选项菜单
    ↓
选择"更换头像"类型
    ↓
调起图片选择器（相册/拍照）
    ↓
图片裁剪编辑（正方形比例）
    ↓
预览确认
    ↓
上传图片 → 显示加载状态
    ↓
上传成功 → 全平台同步更新
    ↓
展示成功提示
```

### 3.5 头像展示规则

| 页面 | 头像类型 | 展示位置 | 尺寸 |
|------|----------|----------|------|
| 首页 | 双人头像 | 顶部中央 | 120×120px |
| 个人中心 | 个人头像 | 用户信息区 | 100×100px |
| 时光轴 | 双人头像 | 页面顶部 | 80×80px |
| 默契考验 | 双人头像 | 对战区域 | 60×60px |
| 相册页 | 双人头像 | 页面顶部 | 80×80px |
| 恋爱剧本 | 双人头像 | 页面顶部 | 80×80px |
| 成长任务 | 双人头像 | 页面顶部 | 80×80px |
| 心动信箱 | 双人头像 | 页面顶部 | 80×80px |
| 设置页 | 个人头像 | 设置项左侧 | 60×60px |

### 3.6 异常处理

| 异常情况 | 处理方式 |
|----------|----------|
| 图片格式不支持 | Toast提示："请上传JPG或PNG格式的图片" |
| 图片超过大小限制 | Toast提示："图片大小不能超过XMB" |
| 上传失败 | Toast提示："上传失败，请检查网络后重试" |
| 裁剪取消 | 返回上一级，不做任何操作 |
| 网络中断 | 保存草稿，网络恢复后自动重试 |

## 四、个人主页背景图自定义

### 4.1 功能概述
用户可自定义个人中心页面的背景图，打造个性化空间。

### 4.2 背景图规格

| 属性 | 规格说明 |
|------|----------|
| 建议尺寸 | 750×400 像素（2:1比例） |
| 格式支持 | JPG、PNG |
| 文件大小 | 不超过 3MB |
| 显示方式 | 顶部覆盖显示，渐变过渡 |
| 默认背景 | 系统预设渐变背景（粉色系） |

### 4.3 功能入口设计

#### 入口：个人中心 - 背景区域
```
位置：个人中心页顶部背景区域
交互：
  - 长按背景区域 或
  - 点击背景区域右下角的"编辑"图标
    ↓
弹出选项：
  - 更换背景图
  - 使用默认背景
  - 从相册选择
  - 拍照
  - 取消
```

#### 备选入口：设置页
```
位置：个人中心 → 设置 → 个性化设置
路径：个人中心 > 设置 > 个性化 > 主页背景
```

### 4.4 背景图设置流程

```
用户触发背景设置
    ↓
弹出选项菜单
    ↓
选择图片来源（相册/拍照）
    ↓
图片选择/拍摄完成
    ↓
进入裁剪界面（支持缩放、移动）
    ↓
确认预览
    ↓
上传并应用
    ↓
个人中心背景实时更新
```

### 4.5 背景图展示效果

```
┌─────────────────────────────┐
│      [背景图区域]            │
│         750×400             │
│    ┌─────────────────┐      │
│    │   渐变遮罩层     │      │
│    │  (底部30%区域)   │      │
│    └─────────────────┘      │
│                             │
│       [双人头像]            │
│       [用户昵称]            │
│       [配对信息]            │
├─────────────────────────────┤
│                             │
│      [功能列表区域]          │
│                             │
└─────────────────────────────┘
```

### 4.6 背景图管理

| 功能 | 说明 |
|------|------|
| 更换背景 | 支持随时更换，新背景覆盖旧背景 |
| 恢复默认 | 一键恢复系统预设背景 |
| 历史记录 | 保留最近3张使用过的背景图 |
| 推荐背景 | 提供节日主题、季节主题等推荐背景 |

## 六、交互设计规范

### 6.1 头像上传交互

#### 点击反馈
- 头像区域点击后，0.2s内显示高亮反馈
- 弹出底部操作菜单，带轻微上滑动画

#### 上传状态
```
选择图片后：
  - 显示裁剪界面，默认1:1比例
  - 支持双指缩放、单指拖动
  - 提供"取消"和"确认"按钮

上传过程中：
  - 显示圆形进度指示器
  - 头像区域显示半透明遮罩
  - 文字提示："上传中..."

上传成功：
  - Toast提示："头像更新成功"
  - 新头像淡入显示（300ms过渡动画）

上传失败：
  - Toast提示具体错误原因
  - 保留原头像，提供"重试"按钮
```

### 6.2 背景图设置交互

#### 触发方式
- 长按背景区域1秒触发
- 或点击右下角悬浮编辑按钮

#### 编辑界面
```
底部弹出半屏菜单：
  ┌─────────────────────┐
  │  更换主页背景        │
  ├─────────────────────┤
  │  📷 拍照            │
  │  🖼️ 从相册选择      │
  │  🎨 推荐背景        │
  │  🔄 恢复默认        │
  ├─────────────────────┤
  │  取消               │
  └─────────────────────┘
```

### 6.3 品牌元素展示规范

#### Logo使用
- 主Logo：潘瑶星球 + 星球图标
- 副标题：记录我们的爱情

## 七、技术实现要点

### 7.1 头像存储方案
- 使用云存储服务（如阿里云OSS、腾讯云COS）
- 图片上传后生成多尺寸缩略图
- CDN加速，确保加载速度

### 7.2 数据同步
- 头像更新后，通过WebSocket推送变更通知
- 各页面监听头像变更事件，实时更新
- 本地缓存头像URL，减少重复加载

### 7.3 性能优化
- 头像图片懒加载
- 使用WebP格式，减少文件大小
- 缓存策略：头像图片缓存7天

## 八、验收标准

### 8.1 功能验收

| 验收项 | 验收标准 |
