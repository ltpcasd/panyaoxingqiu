## 一、配色方案优化

### 1.1 主色调优化（更温暖）

| 名称 | 优化前 | 优化后 | 说明 |
|------|--------|--------|------|
| 主色 | `#FF9A8B` | `#FF7B7B` | 更温暖的珊瑚粉 |
| 主色浅 | `#FFB4A8` | `#FFA8A8` | 柔和蜜桃色 |
| 主色深 | `#E8887A` | `#E86565` | 加深情情感 |
| 主色渐变 | 单向渐变 | 多层暖色渐变 | 增加层次感 |

### 1.2 新增暖色辅助色

```scss
// 新增温暖色系
$warm-coral: #FF6B6B;      // 珊瑚红 - 强调温暖
$warm-peach: #FFB7B2;      // 蜜桃粉 - 柔和过渡
$warm-cream: #FFF0E5;      // 奶油白 - 背景温暖
$warm-rose: #FFD1D1;       // 玫瑰粉 - 点缀装饰
$warm-gold: #FFB347;       // 暖金色 - 闪耀点缀
```

### 1.3 渐变优化（更温暖）

```scss
// 主渐变 - 温暖日出
$primary-gradient: linear-gradient(135deg, #FF7B7B 0%, #FF9A8B 50%, #FFB7B2 100%);

// 爱心渐变 - 甜蜜组合
$love-gradient: linear-gradient(135deg, #FF6B6B 0%, #FFB7B2 100%);

// 金色渐变 - 温暖阳光
$gold-gradient: linear-gradient(135deg, #FFB347 0%, #FFD93D 100%);

// 背景渐变 - 奶油温暖
$bg-gradient: linear-gradient(180deg, #FFF5F0 0%, #FFF0E5 50%, #FFEBE0 100%);
```

## 三、图标风格优化

### 3.1 图标设计原则

- **风格**：现代线性图标 + 圆润设计
- **线宽**：2px 粗线条，视觉更舒适
- **圆角**：所有转角圆润处理
- **色彩**：渐变填充或单色暖色调

### 3.2 核心图标替换方案

| 功能模块 | 优化前 | 优化后 | 图标描述 |
|----------|--------|--------|----------|
| 时光轴 | 📅 | ⏱️ → 渐变时钟图标 | 圆润沙漏+渐变填充 |
| 记忆星云 | 📷 | 💫 → 渐变星云图标 | 星星+渐变填充 |
| 默契考验 | 🎯 | 🎲 → 渐变骰子图标 | 爱心骰子+渐变 |
| 编辑 | ✏️ | ✎ → 线性编辑图标 | 圆润铅笔轮廓 |
| 相机 | 📷 | ◉ → 线性相机图标 | 圆润相机轮廓 |
| 通知 | 🔔 | ⋱ → 线性铃铛图标 | 圆润铃铛轮廓 |
| 设置 | ⚙️ | ⚙ → 渐变齿轮图标 | 圆润齿轮+渐变 |
| 关于 | 📋 | ⓘ → 线性信息图标 | 圆润信息圈 |

### 3.3 图标样式规范

```scss
// 图标容器
.icon-wrapper {
  width: 48px;
  height: 48px;
  border-radius: 14px;
  display: flex;
  align-items: center;
  justify-content: center;
  background: linear-gradient(135deg, rgba(255, 123, 123, 0.15) 0%, rgba(255, 183, 178, 0.15) 100%);
}

// 渐变图标
.icon-gradient {
  font-size: 24px;
  background: linear-gradient(135deg, #FF7B7B 0%, #FFB7B2 100%);
  -webkit-background-clip: text;
  -webkit-text-fill-color: transparent;
}

// 线性图标颜色
.icon-outline {
  font-size: 22px;
  color: #FF7B7B;
}
```

## 四、卡片/按钮样式优化

### 4.1 卡片样式优化

```scss

@mixin card-warm {
  background: $bg-white;
  border-radius: 20px;  // 更圆润
  box-shadow: 
    0 4px 20px rgba(255, 123, 123, 0.08),  // 粉色阴影
    0 2px 8px rgba(0, 0, 0, 0.04);         // 基础阴影
  border: 1px solid rgba(255, 123, 123, 0.1); // 微妙边框

// 特色卡片（渐变背景）

  background: $primary-gradient;
  border-radius: 20px;

    0 8px 32px rgba(255, 123, 123, 0.3),
    0 4px 16px rgba(255, 123, 123, 0.2);
}
```

